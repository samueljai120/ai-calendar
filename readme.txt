== Description ==

AI Calendar is a modern, feature-rich event management plugin for WordPress. It provides a beautiful and responsive calendar interface with customizable themes and layouts.

Features:

* Beautiful responsive calendar interface
* Multiple theme options and customizable colors
* Event management with detailed information
* Social sharing integration
* Export events to various calendar formats
* Location mapping support
* Customizable templates
* Modern sharing options

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ai-calendar` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->AI Calendar screen to configure the plugin
4. Insert the calendar using shortcode [ai_calendar] or use the block editor

== Frequently Asked Questions ==

= Can I customize the calendar appearance? =

Yes, you can customize colors, layouts, and themes through the theme settings panel.

= Does it support recurring events? =

Yes, you can create recurring events with various patterns.
